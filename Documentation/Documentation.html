A) About the Game - top

Tile Master Match Game requires you to match 3 blocks of the same element, you will pass the level until all tiles are matched.Which is different with Mahjong or other block puzzle game. Tile Master Match consists a large number of levels that get progressively more difficult

B) Features - top

  • 100 levels
  • Inluding 3D arts (.fbx files)
  • One Touch Control
  • Responsive
  • Playable in all platforms (Mobile, PC, etc.)

C) Construct File - top

Consruct 3 files ( .c3p) are included all sources and game mechanics. You can go to editor here.

How to change graphic assets

You can replace all sprites in the images file with images that have the same names. You can also edit the frames in the .c3p file.

How to change puzzle images

Puzzle images composed of 9 small images(120x120). Here is a guide of how to slice images into small pieces using Photoshop

You can replace all puzzle pieces in the images file with images that have the same names. You can again also edit the frames in the .c3p file.

How to change game sounds

You can replace all sounds in the media file with sounds that have the same names. Please make sure they are in m4a/ogg format.

How to add new language

Double click "Languages.json" in the "project" bar



Increase width by 1 and fill in the blanks with your translated texts. Enter your language code in line 0. Check all language codes here


Note that if you export the game for web(HTML5), there will be too many "level.json" files. You can create a folder, name it "levels" and move all the level.json files into that folder if you like. But it's not necessary. The game will work either way.
